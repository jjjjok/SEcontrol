# 二手交易平台

基于springmvc实现的二手交易网站，接入第三方支付接口，完成了作为一个项目的闭环，软件工程基础课设
## 项目要点
 - 基于springmvc设计，使用mysql数据库，使用jdbc操作数据库
 - 基于瀑布模型开发
 - 项目从计划、可行性分析、需求分析、概要设计、详细设计、编码测试这些阶段均有文档，并进行版本控制
 - 接入第三方支付接口，可实现扫码支付功能
 - 具有购物车，支付，订单查询，搜索等商城基本功能
 ## 文件结构
  - 文档，ppt——软件工程基础课程所需要提交与评审的文件
  - koudaiwang-out/——部署上线所需的war包
  - koudaiwang-src-test——单元测试用例
  - koudaiwang-src-main-java-com-SE-bean——实体类
  - koudaiwang-src-main-java-com-SE-controller——业务层
  - koudaiwang-src-main-java-com-SE-dao——数据库dao层
  - koudaiwang-src-main-java-com-SE-port——外部接口层
  - koudaiwang-src-main-java-com-SE-util——工具类
  - koudaiwang-src-main-webapp——前端相关
